# Web Drummer

Web based drum computer built with React

## Requirements

- Node v18
- NPM v9

## Installation

```sh
cd ~
git clone https://github.com/vp9000/web-drummer
cd web-drummer
npm ci
```

## Run dev server

```sh
npm run dev
```

## Build / Serve

```sh
npm run build

## In terminal
npm run serve

## As pm2 process
pm2 start npm --name "web-drummer" -- run serve
```
